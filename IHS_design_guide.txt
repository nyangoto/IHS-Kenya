IHS Kenya Design System & Brand Guide
Executive Summary
Based on my analysis of the IHS Kenya website as a senior UI/UX designer with 15+ years of enterprise experience, I've created a comprehensive design system that reflects their position as a forward-thinking, professional housing investment organization. The design demonstrates enterprise-grade sophistication with clear visual hierarchy and deliberate color psychology.

Recommended: Inter, Segoe UI, or similar humanist sans-serif for enterprise applications

Typography Rules
Headings: Use Dusty Blue or Navy Blue, never lighter than 600 weight
Body Text: Charcoal Gray at 14-16px with 1.5-1.6 line height for readability
Contrast Ratio: Maintain minimum 4.5:1 for WCAG AA compliance
All-caps: Reserved for labels, badges, and button text only
Alignment: Headings justify (left-aligned), body text uses justified alignment for formal sections

10. BRAND VOICE IN DESIGN
Visual Language
Professional yet approachable: Enterprise rigor without coldness
Trustworthy: Navy blue and white create stability
Action-oriented: Red accents drive conversions
Community-focused: Warm imagery of real people and places
Sustainable: Green elements in photography reflect environmental commitment
Dos and Don'ts
✅ DO:

Use generous whitespace for enterprise elegance
Maintain strong color contrast
Keep interactions smooth and purposeful
Use real photography of actual properties
Ensure all CTAs are highly visible
❌ DON'T:

Use busy patterns or textures
Over-saturate colors
Use decorative icons without purpose
Make text small (minimum 14px body)
Create confusing navigation hierarchies
11. ACCESSIBILITY STANDARDS
WCAG 2.1 Level AA: Minimum standard for all components
Color Contrast: 4.5:1 for normal text, 3:1 for large text (18px+)
Focus Indicators: Visible on all interactive elements
Alt Text: Descriptive alt text for all images
Keyboard Navigation: Full keyboard support
Screen Readers: Proper semantic HTML and ARIA labels
12. IMPLEMENTATION GUIDELINES
Code Standards
CSS: BEM naming convention (e.g., .button--primary)
Spacing: CSS variables using 8px units (--space-s: 8px, --space-m: 16px)
Colors: CSS variables for theming (--color-primary, --color-accent)
Typography: Web-safe font stack with fallbacks

14. ENTERPRISE DESIGN PRINCIPLES APPLIED
As someone with 15+ years in enterprise design, here's what I've identified as key to IHS's success:

Trust Through Clarity: Navy blue establishes authority; clean layouts build confidence
Action Through Color: Red buttons cut through noise; high conversion potential
Professional Restraint: Limited palette and simple geometry feel sophisticated
Accessibility-First: Enterprise users include diverse accessibility needs
Scalability: System designed to expand without losing cohesion
Performance: Clean design = faster load times, better user experience
Data Hierarchy: Statistics sections prioritize metrics important to investors
Clear CTAs: Multiple conversion paths (Investor Portal, Learn More buttons)
